Book a discovery peer review

You can use this service to:

  • book an end of discovery peer review
  • provide a summary of the project in discovery
  • update the project details, for example, the name of a discovery

You can book a discovery peer review for your own project, or on behalf of a team

Start now

Before you start

You need the names of your:

  • business partner
  • delivery manager
  • product manager
  • senior responsible officer

You will also need your:

  • project name
  • project code

If you don't know your project code

Find your project code on the . If your project does not have a code, you can still book a peer review and add it later.

Find your project code on the DDaT porfolio tracker (opens intranet page in new tab). If you do not have access to the DfE intranet, ask a colleague with access to get the information for you.

You can check the manage page to see if a request has already been submitted for your discovery.
